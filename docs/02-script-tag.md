# Script Tag

Add the Fathom script to your `<head>` section:

```blade
// torchlight! {"lineNumbers": false}
<!DOCTYPE html>
<html>
<head> <!-- [tl! focus] -->
    <!-- ... -->
    <x-fathom-script/> <!-- [tl! focus] -->
</head> <!-- [tl! focus] -->
<body>
    <!-- ... -->
</body>
</html>
```

The script tag will be automatically rendered respecting your configuration.
